# Industrial Machine Vision – Defect Detection

This project demonstrates a simple machine vision inspection pipeline for detecting surface defects in industrial products using image processing.

The system simulates a production line inspection process using Python and OpenCV.

## Technologies
- Python
- OpenCV
- Image processing

## Applications
- Industrial inspection
- Quality control
- Automated visual inspection

## Future improvements
- Camera calibration
- Real-time image acquisition
- Deep learning based defect detection
